Typography

For a school project on typography, I explored the Bronzo typeface. I created a type specimen using HTML and CSS to showcase the unique characteristics and energy of Bronzo. Additionally, I designed a series of posters in Adobe Illustrator to further explore the expressive style of this typeface.

What is a type specimen?

A type specimen is a visual presentation of a typeface, showing its different styles, weights, and characteristics. It helps designers understand how the font behaves in various layouts. For this project, I created a type specimen that includes animations, combining HTML, CSS, Illustrator, and After Effects.
